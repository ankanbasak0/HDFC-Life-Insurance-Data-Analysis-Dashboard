# HDFC-Life-Insurance-Data-Analysis-Dashboard
Designed and developed an end-to-end interactive Power BI dashboard analyzing 10,000+ insurance policy records across 5 policy types (Travel, Health, Auto, Life, Home). Performed data cleaning and transformation using Power Query, built DAX measures to track KPIs including ₹5.98M in premiums, ₹600.55M in coverage, and ₹16.91M in claim amounts. Visualized claim status distribution (Rejected: 43.5%, Settled: 33.8%, Pending: 22.6%) and policy-wise performance with dynamic slicers for Policy Number, Claim Number, and Customer ID.

Tools: Power BI Desktop · Power Query · DAX · Data Modeling · CSV
