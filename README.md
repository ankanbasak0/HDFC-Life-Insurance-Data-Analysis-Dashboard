# HDFC-Life-Insurance-Data-Analysis-Dashboard
Developed an end-to-end Power BI dashboard analyzing 10,000+ insurance policies across 5 types. Tracked KPIs ₹5.98M premiums, ₹600.55M coverage, ₹16.91M claims and visualized claim status distribution (Rejected 43.5%, Settled 33.8%) using DAX measures and Power Query.
